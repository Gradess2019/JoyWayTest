UE version: 4.26 source
VR headset: Oculus Rift

I totally recommend test the app using batch files:
- client_vr.bat - starts a client in VR mode, default map - JoyWayMap1
- server.bat - starts a server, default map - JoyWayMap1
Just replace UE4Editor.exe with your own path to UE4Editor.exe

Action mappings:
Oculus Touch:
- Trigger - Fire/Select
- Grip - Grab item
- B/Y - Reload weapon

Keyboard:
O - connect to the started server
P - possess DefaultPawn (keymap WASD + Mouse)
R - commit a suicide (player will die, and then respawn)
